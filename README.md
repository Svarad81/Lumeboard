# 🌌 LumeBoard Extension (v1.0)
### A Premium, Glassmorphism Dashboard for Chrome

LumeBoard is a high-performance "New Tab" bookmark manager designed with a state-of-the-art glassmorphism aesthetic. It features industrial-grade 2D grid reordering, dynamic theme-aware accents, and a robust "Digital DNA" attribution system.

---

## 🚀 Quick Start (No Setup Required)
If you just want to use the extension without building from source:

1.  **Download** this repository as a ZIP or clone it.
2.  Open **your browser**.
3.  Navigate to the extensions settings (e.g., `chrome://extensions`).
4.  Enable **Developer Mode** (top-right toggle).
5.  Click **Load Unpacked**.
6.  Select the **`dist`** folder inside the LumeBoard directory.
7.  Open a **New Tab** and enjoy!

---

## 🛠️ Developer Guide (Build from Source)
If you want to modify the code or contribute:

1.  **Clone** the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  **Build the extension**:
    ```bash
    npm run build
    ```
    *This generates a fresh `dist` folder for Chrome.*
